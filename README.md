## Project Overview
This repository contains a simple multi-page HTML5 website for **Sunrise Hotel**, developed as part of IT1713 Assignment 01. The site uses semantic HTML5 elements, external CSS, and includes all required pages and assets.

## Website Pages
- `index.html` — Home (landing) page: welcome banner, featured rooms, special offers
- `rooms.html` — Rooms & Facilities: details of rooms, nightly rates, Book Now buttons, facilities
- `dining.html` — Dining & Services: restaurant, café, bar, menu highlights, additional services
- `testimonials.html` — Testimonials: customer reviews with video/audio and checkboxes
- `contact.html` — Contact Us: address, contact numbers, inquiry form

## Assets
All images are located in the `images/` folder. CSS files are in `css/`. JavaScript (if any) is in `js/`.

## Accessibility & Best Practices
- Semantic HTML5 elements (header, nav, main, section, footer, article) used.
- All `<img>` tags include `alt` attributes.
- Navigation bar is sticky (remains at top while scrolling).
- Contact and subscription forms included in the footer and Contact page.
